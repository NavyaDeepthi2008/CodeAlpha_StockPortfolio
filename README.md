# Stock Portfolio Tracker

## Description
This project is a simple Stock Portfolio Tracker developed using Python.  
It allows users to enter stock names and quantities, then calculates the total investment value using predefined stock prices.

## Features
- User input for stock names and quantity
- Predefined stock price dictionary
- Calculates total investment value
- Optional file saving feature
- Console-based application

## Technologies Used
- Python

## Concepts Used
- Dictionary
- Input/Output
- Basic Arithmetic
- File Handling

## How to Run
1. Install Python on your system.
2. Open terminal or command prompt.
3. Run the following command:

```bash
python stock_tracker.py
```
